Crear un producto

Aprende a registrar productos con SKU, precio, impuestos e imagen.

Pasos para crear un producto

  1. Ve a Catálogo → Productos en el menú lateral.
  2. Haz clic en + Nuevo producto.
  3. Llena los campos obligatorios: Nombre, SKU, Precio.
  4. Asigna una categoría (Bebidas, Alimentos, etc.).
  5. Define el IVA aplicable (16%, 8%, 0% o exento).
  6. Sube una imagen (opcional pero recomendado para POS visual).
  7. Guarda con Crear producto.
Formulario de nuevo producto

Campos avanzados

  • Código de barras: permite escanear el producto al cobrar.
  • Stock inicial: cantidad disponible al crear el producto.
  • Stock mínimo: alerta de reabasto.
  • Costo: usado para calcular margen y rentabilidad.
Tip: Si tienes muchos productos, usa la importación masiva por CSV.
