KEY FEATURES
============

OTP Generation:
  + 6-digit codes (000000-999999)
  + SHA-256 hashing
  + 10-minute TTL

Email Delivery:
  + Resend API support
  + SendGrid fallback
  + HTML templates
  + Plain text fallback

Security:
  + Rate limiting (5/15min)
  + Max 5 attempts
  + Input validation
  + No hardcoded secrets

DATABASE
========

customer_otps table:
  + email
  + code_hash (SHA-256)
  + expires_at (10 min TTL)
  + ip address
  + attempt tracking
  + consumed_at tracking

email_log table:
  + to_email
  + subject
  + template
  + status
  + provider (resend/sendgrid)
  + provider_id
  + error (if failed)

API ENDPOINT
============

POST /api/auth/send-otp

Request:
  {
    "email": "user@example.com"
  }

Response (200):
  {
    "ok": true,
    "message": "OTP enviado a tu email",
    "expires_in": 600,
    "provider": "resend",
    "provider_id": "email_xxxxx"
  }

Rate Limit: 5 per 15 minutes per IP
Status Codes: 200/400/429/500

ENVIRONMENT SETUP
=================

Optional:
  RESEND_API_KEY=re_xxxxxxxxxxxxx
  RESEND_FROM_EMAIL=noreply@yourdomain.com
