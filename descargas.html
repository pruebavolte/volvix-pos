⚡
PWA · Instalable en cualquier dispositivo

Instalar como PWA Disponible

Instala Volvix POS como aplicación en tu dispositivo. Funciona offline y se siente nativa.

Chrome / Edge (Android, Windows):
Abre Volvix POS → menú (⋮) → "Instalar app"

Safari (iOS, macOS):
Abre Volvix POS → botón compartir → "Añadir a pantalla de inicio"

Firefox (Android):
Abre Volvix POS → menú (⋮) → "Instalar"
