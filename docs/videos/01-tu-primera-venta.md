# Video 1: Tu Primera Venta en 5 Segundos

## Meta
- Duracion objetivo: 30-60s (optimo: 45s)
- Audiencia: cliente nuevo / dueno de negocio / cajero
- Idioma: ES Mexico (formal-casual, energico)
- Music: lo-fi instrumental upbeat, royalty-free (BPM 90-110)
- Aspect ratio: 16:9 desktop (YouTube), 9:16 vertical para reels (TikTok/IG)
- Tono: confiado, dinamico, "esto es facilisimo"

## Setup pre-grabacion
- Browser: Chrome con perfil limpio (sin extensiones visibles)
- Resolucion: 1920x1080 (escalable a 4K si OBS lo soporta)
- Hidden: bookmarks bar oculto (Ctrl+Shift+B), dev tools cerrado
- Tab: solo /salvadorex_web_v25.html visible
- Volumen sistema: 70% (para captar feedback sonoro de la app)
- Mic: Blue Yeti / RODE NT-USB / Shure MV7 con pop filter
- Pantalla: modo claro forzado (NO dark mode para grabacion - mas legible en compresion video)
- Cursor: highlight on (usar Cursor Highlighter o ZoomIt para resaltar clicks)
- Notificaciones: OFF (modo no molestar Windows)
- Zoom navegador: 110% para legibilidad mobile

## Storyboard (escenas con timestamps)

### 0:00-0:03 - Hook (atencion)
**VOICEOVER**: "Cobrar a un cliente en 5 segundos. Te lo demuestro."
**VISUAL**: Logo Volvix splash (1.5s) + texto grande animado "5 SEGUNDOS" con countdown
**ACTION**: Fade-in rapido al POS dashboard
**TEXTO ON-SCREEN**: "Volvix POS" (esquina inf izquierda)
**SFX**: Whoosh transition

### 0:03-0:08 - Buscar producto
**VOICEOVER**: "Buscas el producto..."
**VISUAL**: Cursor mueve a input de busqueda, click, escribe "lat"
**ACTION**: Autocomplete muestra "Latte $45.00" instantaneo
**TEXTO ON-SCREEN**: "Buscar" (flecha apuntando al input)
**SFX**: Tipeo teclado mecanico (sutil)

### 0:08-0:13 - Agregar al carrito
**VOICEOVER**: "Lo agregas con un click..."
**VISUAL**: Click en producto del autocomplete, animacion de salto al carrito derecho
**ACTION**: Carrito muestra "Latte x1 - $45.00", subtotal $45.00 visible
**TEXTO ON-SCREEN**: "+1 producto" (toast superior)
**SFX**: Ding suave de notificacion

### 0:13-0:20 - Cobrar con F12
**VOICEOVER**: "Cobras con F12, sin tocar el mouse..."
**VISUAL**: Tecla F12 highlight on-screen, modal de cobro abre con animacion slide-up
**ACTION**: Modal muestra: Total $45.00, input "Recibido", opcion efectivo
**TEXTO ON-SCREEN**: "F12 = Cobrar" (badge naranja)
**SFX**: Click teclado fuerte

**VOICEOVER**: "Pone $50, calcula cambio..."
**VISUAL**: Tipeo "50" en input recibido
**ACTION**: Sistema calcula instantaneo "Cambio: $5.00" en verde
**TEXTO ON-SCREEN**: "Cambio: $5" (highlight verde)

### 0:20-0:27 - Imprimir ticket
**VOICEOVER**: "...y se imprime el ticket."

### 0:27-0:35 - Beneficio key
