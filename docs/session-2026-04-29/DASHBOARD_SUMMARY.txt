🔌 ENDPOINTS API (11 TOTAL)
================================================================================

1. GET /api/dashboard/sales-summary?period=today|week|month|year
   → Total ventas, transacciones, ticket promedio

2. GET /api/dashboard/sales-by-product?period=month
   → Top 15 productos por ventas

3. GET /api/dashboard/sales-by-category?period=month
   → Agrupación por categoría

4. GET /api/dashboard/sales-by-day?period=month
   → Ventas diarias (para gráficos)

5. GET /api/dashboard/transactions?limit=50&offset=0&filter=all|pending|completed|refunded
   → Historial de transacciones paginado

6. GET /api/dashboard/payment-methods?period=month
   → Desglose por método de pago

7. GET /api/dashboard/users?limit=50&offset=0&role=all&status=all
   → Listado de usuarios

8. GET /api/dashboard/inventory?low-stock-only=false
   → Stock de productos con alertas

9. GET /api/dashboard/top-products?limit=10&period=month
   → Top N productos más vendidos

10. GET /api/dashboard/kpis?period=month
    → Indicadores clave de desempeño

11. GET /api/dashboard/integration-status
    → Estado de servicios externos

Todos requieren: requireAuth() + tenant_id en sesión
