# Twilio Setup
1. console.twilio.com → Account SID + Auth Token (rotar si necesario)
2. Phone Numbers → Buy a number ($1-2/mes para SMS) o usar Sandbox
3. WhatsApp Sandbox: Messaging → Try it out → "join <palabra>"
4. Vercel env: TWILIO_ACCOUNT_SID, TWILIO_AUTH_TOKEN, TWILIO_WHATSAPP_FROM, TWILIO_SMS_FROM
