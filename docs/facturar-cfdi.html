Facturación

Generar factura CFDI 4.0

Emite facturas electrónicas válidas ante el SAT desde el POS.

Requisitos previos

  • Certificado de Sello Digital (CSD) cargado en el sistema.
  • PAC contratado (Volvix soporta Facturama, Finkok, SW Sapien).
  • Cliente con datos fiscales completos.

Facturar una venta existente

  1. Ve a Ventas → Historial.
  2. Encuentra el ticket a facturar.
  3. Pulsa Facturar (icono de documento).
  4. Selecciona o crea el cliente con RFC.
  5. Confirma Uso de CFDI y Forma de pago.
  6. Pulsa Timbrar.
  7. Descarga el XML y PDF, o envíalos por correo.
Modal de timbrado CFDI

Cancelar una factura

Desde Facturación → Emitidas, selecciona la factura y pulsa Cancelar. Indica el motivo SAT (01, 02, 03, 04). El receptor debe aceptar la cancelación si tiene más de 24h.

Atención: A partir de 2026 el SAT exige motivo de cancelación obligatorio.
