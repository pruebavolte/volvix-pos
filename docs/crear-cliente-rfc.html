Clientes

Crear cliente con RFC

Registra clientes con datos fiscales para emitir facturas CFDI.

Datos requeridos para CFDI 4.0

  • RFC (12 caracteres persona moral, 13 persona física).
  • Razón social (debe coincidir EXACTAMENTE con la Constancia de Situación Fiscal).
  • Código postal del domicilio fiscal.
  • Régimen fiscal (catálogo SAT).
  • Uso de CFDI (G03 Gastos en general, P01 Por definir, etc.).

Pasos

  1. Ve a Clientes → Nuevo cliente.
  2. Llena los datos básicos: nombre, email, teléfono.
  3. Activa el switch Datos fiscales.
  4. Captura RFC, razón social, CP y régimen.
  5. Pulsa Validar con SAT (recomendado, evita rechazos).
  6. Guarda.
Tip: Pide al cliente su Constancia de Situación Fiscal en PDF — trae todos los datos exactos que necesitas.
