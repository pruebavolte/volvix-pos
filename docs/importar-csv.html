Importar productos desde CSV

Carga masiva de catálogo usando archivos CSV o Excel.

Formato del archivo

El CSV debe tener encabezados en la primera fila. Las columnas obligatorias son:

nombre,sku,precio,categoria,iva,stock
Coca Cola 600ml,COCA600,18.50,Bebidas,16,120
Sabritas Original,SAB001,15.00,Botanas,16,85

Pasos para importar

  1. Ve a Catálogo → Productos → Importar.
  2. Descarga la plantilla de ejemplo.
  3. Llena la plantilla con tus productos.
  4. Sube el archivo CSV (UTF-8).
  5. Revisa la vista previa: el sistema marcará errores en rojo.
  6. Confirma con Importar todo.
Importante: Si un SKU ya existe, el producto se actualiza en lugar de duplicarse.

Errores frecuentes

  • "Encoding inválido": Guarda el CSV como UTF-8 desde Excel (Archivo → Guardar como → CSV UTF-8).
  • "Categoría no existe": Crea la categoría primero o usa una existente.
