Introduccion

Volvix POS REST API v3.4.0 - Sistema de punto de venta multi-tenant con capacidades de IA. Esta documentacion interactiva cubre los 43 endpoints publicos disponibles. Todas las respuestas son JSON UTF-8.

Base URL

https://api.volvix-pos.com/v1

Formato

application/json

Auth

Bearer JWT

Versionado

Header X-API-Version: 2026-04-25

Authentication

La API usa JWT Bearer tokens. Obten un token con POST /api/login y enviarlo en el header Authorization: Bearer <token>. Tokens expiran en 24h, refresh tokens en 30 dias.

Authorization: Bearer eyJhbGciOiJIUzI1NiIsInR5cCI6IkpXVCJ9...
Multi-tenant: Incluye X-Tenant-Id: <tenant_uuid> en cada request despues del login para seleccionar el tenant activo.

Scopes / Roles
