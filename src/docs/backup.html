Configuración

Respaldos automáticos

Configura backup en la nube para no perder ningún dato.

Tipos de respaldo

  • Local: archivo .zip descargable manualmente.
  • En la nube: automático cada 24h en Volvix Cloud.
  • Externo: Google Drive, Dropbox, S3.

Activar backup en la nube

  1. Ve a Configuración → Respaldos.
  2. Activa el switch Backup en la nube.
  3. Define la frecuencia (diaria, semanal).
  4. Define la hora (recomendado: 03:00 AM).
  5. Pulsa Guardar.

Restaurar un respaldo

  1. Ve a Configuración → Respaldos → Historial.
  2. Selecciona el respaldo a restaurar.
  3. Pulsa Restaurar.
  4. Confirma con tu contraseña de dueño.
Importante: restaurar un respaldo sobrescribe los datos actuales. Haz un backup previo antes de restaurar.
