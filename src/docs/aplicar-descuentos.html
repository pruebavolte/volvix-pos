Aplicar descuentos

Descuentos por porcentaje, monto fijo o cupones promocionales.

Tipos de descuento

  • Por porcentaje: ej. 10% sobre todo el carrito.
  • Por monto fijo: ej. $50 menos en la compra.
  • Por producto: aplica solo a un ítem específico.
  • Cupón: código promocional (ej. BIENVENIDA).

Aplicar descuento manual

  1. Con productos en el carrito, pulsa el botón % o presiona F4.
  2. Elige tipo: porcentaje o monto.
  3. Captura el valor y la razón (auditable).
  4. Confirma. El total se recalcula automáticamente.
Permisos: los cajeros pueden necesitar autorización del gerente para descuentos >15%. Configúralo en Roles.

Cupones automáticos

Crea cupones reutilizables en Promociones → Cupones. Define:

  • Código (ej. NAVIDAD2026).
  • Vigencia (fecha inicio/fin).
  • Usos máximos (1, 100, ilimitado).
  • Monto mínimo de compra.
