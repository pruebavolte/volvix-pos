Caja

Abrir caja con fondo inicial

Inicia el turno declarando el fondo en efectivo de tu caja.

Pasos

  1. Inicia sesión con tu usuario de cajero.
  2. Ve a Caja → Abrir caja.
  3. Captura el fondo inicial (efectivo con el que arrancas).
  4. Selecciona la caja física (si tienes varias).
  5. Pulsa Abrir turno.
Modal de apertura de caja

¿Por qué declarar fondo?

El fondo inicial es el efectivo con el que comienza el turno (ej. $500 para dar cambio). Al cerrar la caja, este monto se resta del total para calcular las ventas reales del día.

Importante: No se pueden registrar ventas si la caja no está abierta. El POS bloqueará el botón de cobro.
